Este proyecto corresponde al Laboratorio 03 de la asignatura.
Se implementa una aplicación en Kotlin usando Jetpack Compose (Material 3), donde se muestran ejemplos de contenedores y controles básicos.

Archivo principal

Toda la implementación se encuentra en:

app/src/main/java/com/example/lab03/MainActivity.kt

Funcionalidades implementadas

TopAppBar: barra superior con título.
FloatingActionButton: botón flotante que muestra un Snackbar.
LazyColumn: lista vertical de rutinas.
LazyRow: lista horizontal de categorías.
FlowRow: disposición fluida de metas.
OutlinedTextField: campo de texto para ingresar nombre.
Checkbox: activar o desactivar recordatorios.
Slider: seleccionar la intensidad de la rutina.
AlertDialog: ventana de confirmación.
ProgressBar: barra de progreso de la rutina.
Icon: uso de un ícono representativo.

Ejecución

Clonar el repositorio con: git clone https://github.com/jhimydev-rx/Lab03-AE.git

Abrir el proyecto en Android Studio.

Ejecutar en un dispositivo físico o emulador Android.
